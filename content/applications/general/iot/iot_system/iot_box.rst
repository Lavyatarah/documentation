
=======
IoT Box
=======

The IoT Box is a credit card-sized micro-computer with an IoT program pre-installed. 
The IoT Box is a plug-and-play device that can be connected to the network via Wi-Fi or Ethernet. 
It is used to connect devices to the Odoo database.

.. seealso::
   `Order an IoT box <https://www.odoo.com/app/iot>`_
